Hip-Hop & R&B Editorial Platform\
Full-Stack Web Project | Tech: Node.js, Express, EJS, Bootstrap, JavaScript

Overview\
This is a content-driven editorial website designed to publish articles about hip-hop and R&B. The platform demonstrates full-stack development skills, including server-rendered dynamic pages, modular templating, and responsive design optimized for readability.

Features\
-Dynamic Content Management: Edit articles in place via modal-based forms without leaving the page.\
-Reusable EJS Partials: Modular templates maintain consistent structure and simplify future development.\
-RESTful Routes: Implemented PATCH requests for clean and maintainable content updates.\
-Responsive Design: Mobile-first layout ensures readability across devices.\
-Scalable Architecture: Structured to support ongoing publishing with minimal additional work.\

Tech Stack\
-Backend: Node.js, Express\
-Frontend: EJS, Bootstrap, JavaScript\
-Development Practices: Modular templating, RESTful routing, responsive design\

What I Learned / Highlights\
-Built a maintainable full-stack web app with dynamic server-side rendering.\
-Learned how to integrate in-place editing and modular templating to streamline content updates.\
-Focused on UX and responsiveness for real-world publishing scenarios.\
